## Goal Statement
Maya will initiate or respond to peer greetings (verbal or gesture-based) with 80% accuracy across 10 trials in a variety of settings (classroom arrival, lunch, specials, free play).

**Target Criterion:** 80% accuracy across 10 trials  
**Measurement Type:** Trial Percentage

---

## Implementation Instructions
Record each greeting attempt in the table below. Mark the outcome based on the prompt level required for the student to complete the greeting. 

**Weekly Calculation:**  
$\text{(Total Independent Responses / Total Trials)} \times 100 = \text{Weekly Percentage}$

### Outcome Key
| Code | Meaning | Description |
| :--- | :--- | :--- |
| **I** | Independent | Completed without prompts |
| **VP** | Verbal Prompt | Required a spoken reminder or hint |
| **MP** | Model Prompt | Required a demonstration of the greeting |
| **No** | No Response | Did not respond or incorrect response |
